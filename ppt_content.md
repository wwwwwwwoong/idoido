# 🎨 IDO (I-DO) 서비스 소개 자료

이 문서는 IDO 서비스의 PPT 제작을 위한 핵심 내용을 정리한 자료입니다.

---

## 1️⃣ 서비스 개요 (Service Overview)

### **서비스명**: IDO (아이두)
### **슬로건**: "상상이 현실이 되는 시간, 내 손으로 만드는 동화"
### **핵심 컨셉**:
단순한 동화 읽기를 넘어, **아이가 직접 그린 그림**이 동화 속 주인공이 되는 **참여형 AI 창작 플랫폼**입니다. 아이들의 낙서(Doodle)를 고품질 일러스트로 변환하고, 이를 바탕으로 세상에 하나뿐인 이야기를 만들어냅니다.

---

## 2️⃣ 핵심 사용자 경험 (User Flow)

사용자는 마법사가 되어 다음 4단계 과정을 통해 동화를 완성합니다.

1.  **마법 재료 선택 (Mixer)**
    *   이야기의 배경, 사건, 분위기(Mood)를 직접 선택하여 이야기의 뼈대를 만듭니다.
    *   *예: "우주"에서 "보물찾기"를 하는 "신나는" 이야기*
2.  **캐릭터 창작 (Draw & Create)**
    *   **드로잉**: 아이가 캔버스에 직접 주인공을 그립니다.
    *   **AI 변환**: 삐뚤빼쭐한 낙서가 전문가 수준의 캐릭터 일러스트로 변신합니다.
3.  **스토리 생성 (Story Generation)**
    *   선택한 재료와 생성된 캐릭터를 결합하여 AI가 즉석에서 동화 텍스트와 삽화를 생성합니다.
4.  **출판 및 감상 (Publishing)**
    *   완성된 동화는 **웹툰 모드** 또는 **책 넘김 모드**로 감상할 수 있으며, '내 서재'에 저장됩니다.

---

## 3️⃣ 주요 기능 및 차별점 (Key Features)

### 🎨 1. AI 시각적 추론 (Visual Reasoning)
*   **단순 생성 아님**: 기존의 랜덤 이미지 생성이 아닙니다.
*   **Vision AI 분석**: 아이가 그린 그림의 형태, 색상, 소품(모자, 지팡이 등)을 AI(GPT-4 Vision/Gemini)가 정밀 분석하여, 원작의 의도를 그대로 살린 **고품질 캐릭터**로 재탄생시킵니다.

### 🎭 2. 스타일 생성 엔진 (Style Engine)
*   하나의 캐릭터를 다양한 화풍으로 변환 가능합니다.
    *   따뜻한 동화책 스타일 (Storybook)
    *   클레이 애니메이션 (3D Clay)
    *   수채화 (Watercolor) 등

### 📖 3. 멀티 뷰어 시스템
*   모바일/태블릿 환경에 최적화된 UI 제공.
*   세로 스크롤 방식의 **웹툰 뷰**와 실제 책을 넘기는 듯한 **북 뷰**를 모두 지원.

## 4️⃣ 기술 스택 (Technology Stack)

최신 웹 기술과 생성형 AI 모델을 결합하여 구축되었습니다.

| 구분 | 기술 (Tech) | 비고 |
| :--- | :--- | :--- |
| **Frontend** | **Next.js 15 (App Router)** | 최신 React 프레임워크 |
| **Language** | **TypeScript** | 안정적인 타입 시스템 |
| **Styling** | **Tailwind CSS v4** | 모던하고 빠른 UI 디자인 |
| **Animation** | **Framer Motion** | 부드러운 인터랙션 및 화면 전환 |
| **AI (Logic)** | **OpenAI GPT-4 / Gemini** | 스토리 생성 및 그림 분석 (Vision) |
| **AI (Image)** | **DALL-E 3 / Google Imagen** | 고품질 이미지 생성 |
| **Backend** | **Supabase** | 인증(Auth) 및 데이터베이스 |
| **Database** | **PostgreSQL (via Prisma)** | 데이터 관리 (ORM 사용) |
| **Cloud** | **Vercel** | 글로벌 배포 및 호스팅 |

---

## 5️⃣ 기대 효과 (Vision)

*   **창의력 교육**: 아이 스스로 스토리를 구상하고 그림을 그리며 창의력 발달.
*   **성취감 부여**: 자신의 낙서가 멋진 작품이 되는 경험을 통해 자존감 향상.
*   **가족 유대감**: 부모와 아이가 함께 이야기를 만들며 소통하는 매개체.
